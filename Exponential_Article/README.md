# ThesisJuan
Repository of simulation files for the Thesis

As of 10/12, the files used are:

Observe_Interval: File where I define the functions of the variance and intervals.

ObservationsEstimations: File where the parameters for the experiments are set and saved in a .csv file.

simulation: File where a simulation with some type of outlier is generated to observe the distribution graphically.

estimation: File where the estimation functions using DPD and MLE are stored.

derivatives: File where functions that calculate the derivative and the Hessian of the minimization problem for the empirical DPD are stored.

Table_Confidence_Interval: reads csv and creates a table

Observe_MSE: Reads the .csv file where I store the simulation data and displays the MSE of the estimates for different values of beta and outlier proportions.
